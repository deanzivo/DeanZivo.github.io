续篇

Posted on 2019-04-07 | In 网站技术

上一篇写到了安装hexo，又装了Next主题，今天基本上把Next搞定了大部分。

总结一下：Hexo和Next其实并不复杂，很多官方的资料可以先去看一看。

Hexo的官指文章：https://hexo.io/zh-cn/

Next的官指文章：http://theme-next.iissnan.com

以上官指基本上非常详细了，有一些技术问题随时百度一下也就ok了。

我的Next就是安装了侧边栏，用的是Pisces主题。

用哪个就把前面的#去掉就可以了。

我又弄了增加 标签和分类，也很简单就照做就行了。注意的是，标签可一个有多个标签，而分类只有一个。 标签的分类用 - 这个( 空格-空格 ) 分开就行。

接下来是在标准页里面去添加标签和分类，打开scaffolds/post.md文件，在tages:上面加入categories:,保存后，之后执行hexo new 文章名命令生成的文件，页面里就有categories:项了。

最后有几个实用的代码：

< !--more—> 用来进行阅读更多的。

· 这个是英文下的 ` 号（数字1 签名的那个，~下面的），一个是代码行，三个是代码高亮。打完三个···后，输入的是代码的语言，然后回车，就直接是代码块了。

写在最后： 标签分割用 - 号，记着是用下面的方式，不然会报错！

# hexo # Typora # Next
