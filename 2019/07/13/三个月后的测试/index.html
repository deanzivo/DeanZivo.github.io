时隔三个月之久，又想起hexo的时候，已经忘了怎么操作了。

留存的记忆已经很难回忆起来，很像老年痴呆的最初级现象—近期遗忘症，30年前的事情还能记得很清楚，而三个月前的就很模糊了，年龄再大点可能就是3天或者3小时了。

现转正题，复习一下，hexo的几个操作，以便下次忘了及时在看一下。

macos还是很稳定的，基本上不需要再系统上做什么调整。

hexo new “postName”#新建文章-在source文件夹_posts的文件夹

hexo new page “pageName”#新建页面

hexo generate#生成静态页面至public目录

hexo server#开启预览访问端口（默认端口4000，'ctrl + c'关闭server）

hexo deploy#将.deploy目录部署到GitHub

hexo help# 查看帮助

hexo version`#查看Hexo的版本

用Typora编辑 _posts文件夹里的 .md 文件

我的零基础hexo之路(续篇)
